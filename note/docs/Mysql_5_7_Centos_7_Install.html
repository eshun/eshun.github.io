    $ mysql5.7的安装与配置(centos7环境)

    $ 第一步：下载mysql

    [root@iZwz90ggnjrdxf19csd8ryZ ~]# wget http://mirrors.sohu.com/mysql/MySQL-5.7/mysql-5.7.17-linux-glibc2.5-x86_64.tar.gz
    1

    $ 1:检查是否本地已经安装了mysql

    rpm -qa | grep mysql
    1

    $ 2:卸载以前的mysql

    rpm -e 已经存在的MySQL全名
    1

    $ 3:安装mysql依赖

    yum install vim libaio net-tools
    1

    $ 第二步：解压文件

    [root@iZwz90ggnjrdxf19csd8ryZ ~]# tar -zxvf mysql-5.7.17-linux-glibc2.5-x86_64.tar.gz -C /usr/local/
    1

    $ 文件名修改为mysql：

    [root@iZwz90ggnjrdxf19csd8ryZ ~]# cd /usr/local/
[root@iZwz90ggnjrdxf19csd8ryZ local]# mv mysql-5.7.17-linux-glibc2.5-x86_64/ mysql
    1
    2

    $ 第三步：配置启动文件

    $ 1、然后去到mysql的support-files目录下,复制my.cnf到 /etc/my.cnf(mysqld启动时自动读取)

    [root@iZwz90ggnjrdxf19csd8ryZ local]# cd mysql/support-files/
[root@iZwz90ggnjrdxf19csd8ryZ support-files]# cp my-default.cnf /etc/my.cnf
cp: overwrite ‘/etc/my.cnf’? yes
    1
    2
    3

    $ 2、配置数据库编码，添加以下内容

    [root@iZwz90ggnjrdxf19csd8ryZ support-files]# vim /etc/my.cnf
[mysql]
default-character-set=utf8

[mysqld]
default-storage-engine=INNODB
character_set_server=utf8
    1
    2
    3
    4
    5
    6
    7

    $ 3、复制mysql.server到/etc/init.d/目录下(目的想实现开机自动执行效果)

    [root@iZwz90ggnjrdxf19csd8ryZ support-files]# cp mysql.server /etc/init.d/mysql
    1

    $ 4、修改/etc/init.d/mysql参数，修改以下内容

    [root@iZwz90ggnjrdxf19csd8ryZ support-files]# vim /etc/init.d/mysql
basedir=/usr/local/mysql
datadir=/usr/local/mysql/data
    1
    2
    3

    $ 5、出于安全便利，创建一个操作数据库的专门用户

    建立一个mysql的组:

    [root@iZwz90ggnjrdxf19csd8ryZ support-files]# groupadd mysql
    1

    建立mysql用户，并且把用户放到mysql组:

    [root@iZwz90ggnjrdxf19csd8ryZ support-files]# useradd -r -g mysql mysql
    1

    给mysql用户设置一个密码:

    [root@iZwz90ggnjrdxf19csd8ryZ support-files]# passwd mysql
    1

    给目录/usr/local/mysql 更改拥有者:

    [root@iZwz90ggnjrdxf19csd8ryZ support-files]# chown -R mysql:mysql /usr/local/mysql/
    1

    $ 第四步：初始化 mysql 的数据库

    [root@iZwz90ggnjrdxf19csd8ryZ support-files]# cd /usr/local/mysql/bin/
[root@iZwz90ggnjrdxf19csd8ryZ bin]# ./mysqld --initialize --user=mysql --basedir=/usr/local/mysql --datadir=/usr/local/mysql/data
    1
    2

    初始化后会生成一个临时密码 root@localhost:：*(最好先记录这个临时密码)

    $ 2.给数据库加密

    [root@iZwz90ggnjrdxf19csd8ryZ bin]# ./mysql_ssl_rsa_setup --datadir=/usr/local/mysql/data
    1

    $ 3.启动mysql

    [root@iZwz90ggnjrdxf19csd8ryZ bin]# ./mysqld_safe --user=mysql &
    1

    $ 4.检查mysql是否启动

    [root@iZwz90ggnjrdxf19csd8ryZ bin]# ps -ef|grep mysql
    1

    发现有进程便代表启动成功。

    $ 第五步：进入客户端

    $ 1.登录:

     [root@iZwz90ggnjrdxf19csd8ryZ bin]# ./mysql -uroot -p
    1

    Enter password:这里输入之前的临时密码

    $ 2.修改密码

    mysql> set password=password('新密码');
    1

    $ 第六步：设置远程访问

    $ 1:打开mysql的默认端口3306:

    [root@iZwz90ggnjrdxf19csd8ryZ bin]# firewall-cmd --zone=public --add-port=3306/tcp --permanent
success
[root@iZwz90ggnjrdxf19csd8ryZ bin]# firewall-cmd --reload
success
    1
    2
    3
    4

    $ 2:设置mysql的远程访问

    设置远程访问账号:grant all privileges on . to 远程访问用户名@’%’ identified by ‘用户密码’;

    mysql> grant all privileges on *.* to root@'%' identified by 'root';
    1

    刷新:

    mysql> flush privileges;
    1

    重起mysql服务:

    service mysqld restart
    1

    $ 第七步：设置开机自启动

    $ 1、添加服务mysql

    [root@iZwz90ggnjrdxf19csd8ryZ bin]# chkconfig --add mysql
    1

    $ 2、设置mysql服务为自启动

    [root@iZwz90ggnjrdxf19csd8ryZ bin]# chkconfig mysql on
    1

    $ 第八步：配置环境变量

    [root@iZwz90ggnjrdxf19csd8ryZ ~]# vim /etc/profile
    1

    最后一行添加:

    export PATH=$JAVA_HOME/bin:/usr/local/mysql/bin:$PATH
    1

    使修改生效:

    [root@iZwz90ggnjrdxf19csd8ryZ ~]# source /etc/profile
    1

    至此,mysql5.7的安装就完成了!!!
